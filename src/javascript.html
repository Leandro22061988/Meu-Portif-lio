Introdução ao JavaScript

O JavaScript é uma linguagem de programação amplamente utilizada para criar páginas web interativas e dinâmicas. Com ele, você pode adicionar comportamentos complexos e interatividade aos seus sites.

Sintaxe do JavaScript

O JavaScript possui uma sintaxe simples e flexível. Aqui está um exemplo básico:

function saudacao() {
  alert('Olá, mundo!');
}
saudacao();
